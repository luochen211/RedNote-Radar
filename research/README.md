# Research Assets

This directory contains a cleaned, git-friendly export of the local research materials that support the hotel video prediction and analysis pipeline.

Included:
- source code
- notebooks
- small sample data
- lightweight configs and metadata

Excluded from this tracked export:
- virtual environments
- IDE metadata
- nested `.git` directories
- cache folders
- model checkpoints and binary weights that are too large for GitHub
- Windows DLL / EXE runtime bundles
- wheel packages

Structure:
- `prediction-pipeline/`: local/global engagement prediction and upstream feature extraction scripts
- `oriental-aesthetics/`: six-metric Eastern color aesthetics and culture tendency demo
- `analysis-metrics/`: remaining analysis notebooks plus audio-emotion and DeepSentiBank references
